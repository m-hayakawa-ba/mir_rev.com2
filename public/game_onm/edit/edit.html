陰陽童子物語　→　エディットモード　→　追加マップについて


　ＭＩＲのゲームをプレイしてくださった方にお願いして、
自作のエディットマップを提供していただきました。
ＭＩＲの作成したマップとは一味違った面白い面がそろっていますので、
ぜひ、ダウンロードして遊んでみてください。

ダウンロードしたzipファイルを解凍し、
出てきたmpfファイルを、陰陽童子物語の、
savedata_mapフォルダに入れれば、遊ぶことが出来ます。

掲載されているマップについては、白、黒ともにクリア確認されていますが、
初期のマップ（No.100前後より昔のもの）については、
リプレイ機能が無かった事もあり、
クリア確認が取れていないマップもあります（アバウトな管理ですまぬ…）。

クリア不能の疑いがかかっているマップは、注意書きがありますので、
各マップのコメント欄もあわせてご覧ください。

　

現在、エディットマップの募集は休止しております。
（管理がめんどくさくなったため・・・許してくれいッ！
　どうしても見てほしい！　という力作ができた方は一応メール下さい！）

　

今までマップを送ってくださった方々へ。

当初、エディットマップを募集し始めた時は、
まさかこれほどたくさんのマップを送っていただけるとは、思ってもいませんでした。
追加マップのコーナーがこれほど充実したものになったのは、
マップを送ってくださった皆様のおかげです。

あらためてお礼を述べさせていただきます。
どうもありがとうございました。
